# Chromeウェブストア掲載用説明文

## 短い説明（132文字以内）

開いているすべてのタブのタイトルとURLを一括取得し、カスタマイズ可能なテンプレート形式でクリップボードにコピーできる拡張機能です。

---

## 詳細な説明

### Tab URL Exporter - タブ情報を簡単にエクスポート

**Tab URL Exporter**は、現在開いているブラウザウィンドウ内のすべてのタブのタイトルとURLを一括で取得し、カスタマイズ可能な形式でクリップボードにコピーできる便利な拡張機能です。

### 主な機能

✅ **一括取得**
- 現在のウィンドウで開いているすべてのタブの情報を自動取得
- タブ数を表示して、取得対象を確認可能

✅ **カスタマイズ可能なテンプレート**
- 出力形式を自由にカスタマイズ可能
- `%%title%%` と `%%URL%%` のプレースホルダーを使用
- Markdown形式、プレーンテキストなど、お好みの形式に対応
- デフォルト形式: `[%%title%%] %%URL%%`

✅ **区切り文字の設定**
- タブ間の区切り文字を自由に設定可能
- 改行、コンマ、その他の文字に対応
- `\n` と入力すると改行になります

✅ **ワンクリックでコピー**
- 取得した情報をワンクリックでクリップボードにコピー
- 設定の保存・リセット機能付き

### 使用例

- **ブックマークの整理**: 開いているタブのURLを一括で取得してリスト化
- **資料作成**: 参考にしたWebページのタイトルとURLを簡単に記録
- **共有**: 複数のタブ情報をMarkdown形式などで共有
- **バックアップ**: 現在の作業セッションのタブ情報を保存

### 使用方法

1. ブラウザのツールバーにある拡張機能のアイコンをクリック
2. ポップアップが開き、自動的に現在のウィンドウの全タブ情報が取得されます
3. 必要に応じてテンプレートや区切り文字をカスタマイズ
4. 「コピー」ボタンをクリックしてクリップボードにコピー

### プライバシー

この拡張機能は、現在開いているタブの情報のみを取得します。外部サーバーへのデータ送信は一切行いません。すべての処理はローカルで完結します。

### 必要な権限

- **activeTab**: アクティブなタブへのアクセス
- **tabs**: すべてのタブ情報の取得
- **clipboardWrite**: クリップボードへの書き込み

これらの権限は、拡張機能の機能を提供するために必要最小限のものです。
